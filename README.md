# hrdemo-microservice
## How to start
* Start backend microservice first
```
cd hrdemo-svc-minimal
mvn spring-boot:run
```
* Then start ui application
```
cd ui-angular1
npm install
bower install
grunt serve
```
or
```
cd ui-angular2-ag-grid
npm install
npm start
```
or
```
cd ui-react-redux-1
npm install
npm start
```
